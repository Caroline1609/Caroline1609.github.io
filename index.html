Plateforme de réservation Euro express

Travail réalisé pour l’agence « Les Avengers du Dev ». L’objectif : développer les fonctionnalités clés d’un projet client pour qu’elles puissent être intégrées dans l’application finale par le Lead Dev.
📄
Sujet : Développement d’un prototype de site pour réserver des trains hypersoniques depuis Paris vers 6 destinations. Le projet comprenait la création des pages HTML/CSS, la définition des classes Java pour gérer les réservations, le calcul automatique des tarifs selon la classe et l’âge, et le stockage des données en base. L’objectif était de fournir les fonctionnalités clés, prêtes à être intégrées par le Lead Dev dans l’application finale.
🎯
Mission :
  • Travailler en équipe de développeurs pour concevoir le prototype du site.
  • Créer les pages HTML/CSS du site (accueil et réservation).
  • Définir les classes Java pour gérer les trains et les réservations.
  • Mettre en place la logique métier : calcul automatique du prix selon la classe et les réductions d’âge.
  • Concevoir le système de stockage des réservations dans la base de données.
  • Produire des livrables clairs et fonctionnels pour intégration par le Lead Dev dans l’application finale.
👨‍💻
Technologies : HTML, CSS
⏳ 3 jours
